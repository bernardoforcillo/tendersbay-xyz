preparing a contribution to submit back to this repository.

## What contributing means here

You may fork this repository and modify the code **only** to prepare a pull request for
review and possible inclusion by the maintainer. You may not:

- keep or deploy a modified copy for any other purpose,
- publish, distribute, or share a fork beyond the pull request itself,
- reuse any part of this code (modified or not) in another project.

See the **Contribution Exception** section of [license.md](./license.md) for the exact
terms.

## How to submit a contribution

1. Fork the repository.
2. Create a branch and make your change.
3. Open a pull request against `main` describing the change and why it's needed.
4. By submitting a pull request, you agree to the Contributor License Agreement below.
